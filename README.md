 ## Installation
 - Setup AwesomeWM 4.2
 - clone this repo into `~/.config/awesome`

 ## Dependencies
 This setup depends on a few things:
 - amixer (alsa)
 - `Ubuntu Mono` Fonts
 - [feh](https://wiki.archlinux.org/index.php/feh)
 - [dmenu](https://wiki.archlinux.org/index.php/Dmenu)
